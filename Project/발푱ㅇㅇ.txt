저희 조는 딥러닝 분석으로 후보 순위를 예측하는 프로그램을 계획했습니다.

역할은 화면과 같이 역할은 맡아 진행했습니다.
개발 일정은 계획 후 데이터를 수집 그리고 분석 개발후 문서작성 이렇게 진행했습니다.
<>
개요 설명

활용기술은 기본적으로 github를 이용하여 파일을 공유하였고,
계획의 기본단계인 자료수집은 특정사이트의 프로텍트 해지를 위해
selenium을 사용했습니다.
다음으로 웹 크롤링은 속도 향상을 위해서 멀티프로세싱을 사용했습니다.
결과 예측하기 위해 딥러닝 기술인 LSTM,ensemble 을 이용했으며,
수집한 데이터 값들을 DB와 연동하였습니다.
마지막으로 결과를 보여주기위해 Django 웹 서비스를 구축했으며,
결과를 보여주기위해 Chart.js 와 wordcloud를 이용했습니다.

프로젝트 목표 설명 (8 페이지

다음은 요구사항입니다.
데이터 수집하는 과정에서 선거 데이터와 시간이 부족한 관계로 
대체 데이터를 연예인 데이터를 사용하기로 하였습니다.
화면과 같이 출구조사는 음원음반, 설문조사는 미디어, 여론은 네이버 뉴스로 대체하여
데이터를 수집했습니다.

다음 메인화면입니다.(10 페이지
프로젝트 첫 화면에는 해당 프로그램의 관한 개요가 설명되있습니다.
왼쪽을 보시면 목차가 있습니다. 기능을 사용하기전에 각 기술에관한 설명이 있습니다.
그리고 그림밑에 기능 선택목록 또한 있습니다.

다음 페이지>
검색 기능 페이지 입니다. 원하는 검색어를 입력합니다.
해당 뉴스를 네이버 API를 이용하여 분석합니다. 
내용이 없다면 화면과 같이 없다고 나올것이며,
내용이 있다면 해당 내용을 불러온뒤 요약하여 화면과 같이 제목 내용 워드클라우드를 보여줍니다.

검색 결과를 좀 더 자세히 보시면, 요약된 뉴스 내용 해당 단어의 관한 최근 뉴스 10개를 출력하여
목록으로 보여줍니다.

다음 페이지>
다음은 차트입니다. lstm을 이용하여 2018년도부터 2019년도의 데이터를 수집후
chartjs를 이용하여 시각화합니다.
차트 화면에서 데이터가 축적되어가는 과정을 화면을 통해 보여주며, 수집이 완료되면
다음 월의 결과를 예측하여 보여줍니다.
